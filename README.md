# 🚀 Task Buddy | Flutter Productivity App

Task Buddy is a robust, full-stack mobile productivity application built with Flutter and Dart. It features advanced UI components, secure authentication, and complex state management to provide a seamless user experience for managing tasks and engagement.

---

## ✨ Features

*   **🛡️ Secure Authentication**: Integrated OTP authentication flow for secure user access.
*   **🎨 Dynamic Theming**: Smooth theme toggles allowing users to switch between light and dark modes.
*   **🕹️ Integrated Mini-Games**: Built-in interactive modules designed to enhance user engagement.
*   **📁 Category Management**: Advanced state management for creating, viewing, and deleting task categories.
*   **💾 Local Persistence**: Efficient data handling using local database connectivity to save and retrieve user data.

---

## 🛠️ Tech Stack & Skills

*   **Frontend**: Flutter & Dart for cross-platform mobile excellence.
*   **Architecture**: High-complexity, large-scale codebase with single-file scripts reaching up to 5,000 lines.
*   **Logic**: Complex UI modules and rigorous state management for academic evaluation.
*   **Environment**: Developed and maintained within the `insti` project environment.
